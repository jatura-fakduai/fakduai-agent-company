# TOOLS.md - Project Manager

## Agent Communication
- Receive project requests from the human or company lead.
- Send build-ready design requests to Solution Designer.
- Track each agent through shared `STATUS.md` files.
- Escalate unclear scope, missing owners, stale work, and unresolved blockers.

## Key Deliverables
- Implementation plan
- Prioritized backlog
- Agent handoffs
- Risk register
- Status summary with evidence
